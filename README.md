# SCIE-PANC — Site statique

Version HTML/CSS/JS pure. **Aucune dépendance, aucun build requis.**

## Utilisation

Ouvrir `index.html` directement dans un navigateur, ou pousser le dossier sur GitHub Pages.

## Structure

```
index.html          → Accueil (hero + équipe)
project.html        → Description scientifique du projet
news.html           → Actualités (à remplir)
publications.html   → Publications (à remplir)
css/style.css       → Tous les styles
js/main.js          → Nav mobile + animations scroll
images/             → Logo, bannière, drapeaux
```
